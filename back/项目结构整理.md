# 项目结构整理

## 1. 项目概述

这是一个基于Spring Boot的党建管理系统后端项目，采用了MVC架构模式，使用了Spring Security、JWT、Hibernate等技术栈。

## 4. 技术栈

- Spring Boot 3.4.12 - 应用框架
- Spring Security - 安全框架
- JWT (jjwt 0.12.3) - 认证机制
- Hibernate/JPA - ORM框架
- MySQL - 数据库
- Maven 3.8.1+ - 项目构建工具
- Java 17 - 开发语言
- Lombok - 代码简化工具
- ModelMapper 3.1.1 - 对象映射工具
- Volcengine SDK - 火山引擎SDK

## 5. 核心功能模块

1. **用户认证与授权** - 基于JWT的用户认证和角色授权
2. **活动管理** - 活动发布、报名、管理等功能
3. **学习资源** - 学习资料的上传、分类、查询等
4. **新闻资讯** - 党建新闻的发布和展示
5. **公告通知** - 系统公告的发布和管理
6. **支部管理** - 党组织支部的管理
7. **就业服务** - 就业指导、政策、信息等服务
8. **用户画像** - 用户信息和行为的分析和展示
9. **AI功能** - 基于火山引擎的AI功能集成

## 6. 项目特点

- 采用MVC架构，代码结构清晰
- 实现了统一的响应格式和异常处理
- 采用JWT实现无状态认证
- 使用Spring Security进行权限控制
- 提供了丰富的API接口文档
- 包含完整的测试脚本和工具
