# 就业相关接口对接规范

## 1. 接口概述

本文档定义了党建管理系统中就业相关接口的对接规范，包括就业资讯、招聘信息、就业指导、就业政策和就业服务等功能模块的API接口。这些接口主要用于前端展示就业相关信息，所有接口均支持匿名访问。

## 2. 接口列表

| 接口名称 | 请求方法 | 请求路径 | 功能描述 |
|---------|---------|---------|---------|
| 获取就业资讯详情 | GET | /api/employment/detail/{id} | 根据ID获取就业资讯详情 |
| 获取企业招聘信息列表 | GET | /api/employment/recruitment | 获取企业招聘信息列表，支持分页和筛选 |
| 获取就业指导资源列表 | GET | /api/employment/guidance | 获取就业指导资源列表，支持分页和类型筛选 |
| 获取老师指导学生就业情况 | GET | /api/employment/graduate | 获取老师指导学生就业情况列表 |
| 获取就业政策列表 | GET | /api/employment/policies | 获取就业政策列表，支持分页 |
| 获取就业服务列表 | GET | /api/employment/services | 获取就业服务列表 |

## 5. 实体关系

就业相关的数据库实体包括：

1. EmploymentInfo - 就业资讯表
2. EmploymentPolicy - 就业政策表
3. EmploymentGuidance - 就业指导表
4. EmploymentService - 就业服务表
5. TeacherEmploymentGuidance - 老师就业指导表

这些表之间没有直接的外键关系，都是独立的表结构。

## 6. 安全认证

根据SecurityConfig.java中的配置，所有就业相关接口(`/api/employment/**`)均允许匿名访问，无需进行身份认证。

```java
// 允许匿名访问就业服务接口
.requestMatchers("/api/employment/**").permitAll()
```

## 7. 错误处理

所有接口均采用统一的错误处理机制，当接口调用失败时，返回以下格式的错误信息：

```json
{
  "code": "错误代码",
  "message": "错误信息描述"
}
```

常见的错误代码包括：
- 400：请求参数错误
- 404：资源不存在
- 500：服务器内部错误
