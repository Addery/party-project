# 首页推荐活动列表接口对接说明

## 接口概述

该接口用于获取系统首页展示的推荐活动列表，支持自定义返回数量、排序字段和排序方向。

## 接口信息

- **接口地址**：`http://localhost:8080/api/home/activities`
- **请求方式**：`GET`
- **认证方式**：无需认证（已配置匿名访问）

## 请求参数

| 参数名 | 类型 | 必填 | 默认值 | 说明 | 取值范围 |
|--------|------|------|--------|------|----------|
| limit  | Integer | 否 | 6 | 返回活动数量限制 | 1-20 |
| sortBy | String  | 否 | date | 排序字段 | date（日期）、participants（参与人数）、views（浏览量） |
| order  | String  | 否 | desc | 排序方向 | asc（升序）、desc（降序） |

## 响应字段说明

### 活动对象字段

| 字段名 | 类型 | 说明 |
|--------|------|------|
| id | Integer | 活动ID |
| title | String | 活动标题 |
| type | String | 活动类型（固定为"活动"） |
| date | String | 活动日期，格式：yyyy-MM-dd |
| location | String | 活动地点 |
| participants | Integer | 参与人数 |
| views | Integer | 浏览量 |
| status | String | 活动状态：published（已发布）、ended（已结束） |
| image | String | 活动图片URL |
| tags | List<String> | 活动标签，根据分类自动生成 |

## 标签映射规则

| 活动分类 | 对应标签 |
|----------|----------|
| theme | 主题党日 |
| study | 学习教育 |
| volunteer | 志愿服务 |
| democratic | 民主生活 |
| social | 社会实践 |
| training | 培训交流 |
| 其他 | 其他活动 |

## 调用示例

### 默认参数调用

```bash
curl -X GET http://localhost:8080/api/home/activities
```

### 自定义参数调用

```bash
curl -X GET "http://localhost:8080/api/home/activities?limit=3&sortBy=participants&order=desc"
```

## 注意事项

1. 接口返回的数据已按指定条件排序，无需前端再次排序
2. 当请求参数超出限制范围时，系统会自动使用默认值
3. 图片URL可能为占位图，实际部署时需替换为真实图片地址
4. 活动状态仅返回已发布和已结束的活动
