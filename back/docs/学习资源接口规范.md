# 学习资源接口规范说明

## 1. 获取分类列表接口

### 接口功能
获取学习资源的分类列表

### 请求信息
- **请求URL**: `/api/learning/categories`
- **请求方法**: `GET`
- **请求参数**: 无

### 响应信息
- **响应格式**: JSON
- **响应示例**:
```json
{
  "code": 200,
  "message": "获取成功",
  "data": [
    {
      "id": "theory",
      "name": "理论学习"
    },
    {
      "id": "history",
      "name": "党史教育"
    },
    {
      "id": "policy",
      "name": "政策法规"
    },
    {
      "id": "model",
      "name": "先进典型"
    },
    {
      "id": "integrity",
      "name": "廉政教育"
    }
  ]
}
```

## 2. 获取资源列表接口

### 接口功能
获取学习资源列表，支持分页、分类筛选和关键词搜索

### 请求信息
- **请求URL**: `/api/learning/resources`
- **请求方法**: `GET`
- **请求参数**:
  | 参数名 | 类型 | 必填 | 说明 |
  |--------|------|------|------|
  | page | Integer | 否 | 页码，默认为1 |
  | pageSize | Integer | 否 | 每页条数，默认为10 |
  | category | String | 否 | 分类ID，可选值：theory, history, policy, model, integrity |
  | keyword | String | 否 | 搜索关键词，匹配标题和描述 |

### 响应信息
- **响应格式**: JSON
- **响应示例**:
```json
{
  "code": 200,
  "message": "获取成功",
  "data": {
    "resources": [
      {
        "id": 1,
        "title": "习近平新时代中国特色社会主义思想学习纲要",
        "author": "中共中央宣传部",
        "date": "2023-09-15",
        "duration": null,
        "type": "article",
        "level": null,
        "category": "theory",
        "description": "深入学习习近平新时代中国特色社会主义思想的重要辅助读物",
        "image": null,
        "views": 0,
        "rating": 0
      }
    ],
    "page": 1,
    "pageSize": 10,
    "total": 1
  }
}
```

## 4. 获取相关资源接口

### 接口功能
获取与指定资源相关的学习资源推荐

### 请求信息
- **请求URL**: `/api/learning/resources/{id}/related`
- **请求方法**: `GET`
- **请求参数**:
  | 参数名 | 类型 | 必填 | 说明 |
  |--------|------|------|------|
  | id | Integer | 是 | 资源ID |
  | count | Integer | 否 | 返回数量，默认为5 |

### 响应信息
- **响应格式**: JSON
- **响应示例**:
```json
{
  "code": 200,
  "message": "获取成功",
  "data": [
    {
      "id": 2,
      "title": "习近平谈治国理政",
      "category": "theory",
      "image": null
    }
  ]
}
```

- **请求参数**:

  |--------|------|------|------|
  | id | Integer | 是 | 资源ID |

### 响应信息

```json
{
  "code": 200,
  "message": "开始学习成功",

    "recordId": "123456789"
  }

```

## 通用响应说明

### 响应码
| 响应码 | 说明 |
|--------|------|
| 200 | 成功 |
| 400 | 请求参数错误 |
| 404 | 资源不存在 |
| 500 | 服务器内部错误 |

### 错误响应示例
```json
{
  "code": 404,
  "message": "资源不存在"
}
```

## 注意事项

1. 所有接口均需要用户登录后才能访问，请在请求头中携带有效的认证信息
2. 日期格式统一为 `yyyy-MM-dd`
3. 分页参数中，page 从 1 开始计数
4. 接口返回数据中的 `null` 值表示该字段无数据
