# 就业服务接口说明文档

## 1. 文档概述

本文档详细说明了就业服务模块的所有API接口，包括接口路径、请求方法、参数说明、响应格式等信息。所有接口均以`/api/employment`为前缀，支持匿名访问。

## 2. 接口列表

| 接口名称 | 接口路径 | 请求方法 | 功能描述 |
|---------|---------|---------|---------|
| 获取就业统计数据 | `/api/employment/stats` | GET | 获取就业相关统计数据 |
| 获取就业资讯详情 | `/api/employment/detail/{id}` | GET | 获取就业资讯详情 |
| 获取就业资讯列表 | `/api/employment/news` | GET | 获取就业资讯列表 |
| 获取企业招聘信息 | `/api/employment/recruitment` | GET | 获取企业招聘信息列表 |
| 获取就业指导资源 | `/api/employment/guidance` | GET | 获取就业指导资源列表 |
| 获取老师指导学生就业情况 | `/api/employment/graduate` | GET | 获取老师指导学生就业情况列表 |
| 获取就业政策列表 | `/api/employment/policies` | GET | 获取就业政策列表 |
| 获取就业服务列表 | `/api/employment/services` | GET | 获取就业服务列表 |

## 3. 接口详细说明

### 3.1 获取就业统计数据

- **接口路径**: `/api/employment/stats`
- **请求方法**: GET
- **请求参数**: 无
- **响应数据格式**:
  ```json
  {
    "code": 200,
    "message": "获取成功",
    "data": {
      "employmentRate": 90.0,
      "graduateEmploymentRate": 90.0,
      "companyCount": 950,
      "recruitmentCount": 4700
    }
  }
  ```
- **功能描述**: 获取就业相关统计数据，包括本科生就业率、研究生就业率、合作企业数和招聘岗位数。

### 3.2 获取就业资讯详情

- **接口路径**: `/api/employment/detail/{id}`
- **请求方法**: GET
- **请求参数**:
  | 参数名 | 类型 | 必填 | 说明 |
  |-------|------|------|------|
  | id | Integer | 是 | 就业资讯ID |
- **响应数据格式**:
  ```json
  {
    "code": 200,
    "message": "获取成功",
    "data": {
      "id": 1,
      "title": "2024年春季校园招聘会",
      "content": "详细内容...",
      "publishDate": "2024-03-15T00:00:00.000+00:00",
      "source": "学校就业指导中心",
      "viewCount": 1234,
      "category": "招聘信息",
      "createdAt": "2024-03-14T10:00:00.000+00:00",
      "updatedAt": "2024-03-14T10:00:00.000+00:00"
    }
  }
  ```
- **功能描述**: 根据ID获取就业资讯的详细信息。

### 3.3 获取就业资讯列表

- **接口路径**: `/api/employment/news`
- **请求方法**: GET
- **请求参数**:
  | 参数名 | 类型 | 必填 | 说明 |
  |-------|------|------|------|
  | page | Integer | 否 | 页码，默认1 |
  | pageSize | Integer | 否 | 每页数量，默认10 |
  | category | String | 否 | 分类筛选 |
  | isHot | Boolean | 否 | 是否热门资讯 |
- **响应数据格式**:
  ```json
  {
    "code": 200,
    "message": "获取成功",
    "data": {
      "list": [
        {
          "id": 1,
          "title": "2024年春季校园招聘会",
          "publishDate": "2024-03-15T00:00:00.000+00:00",
          "source": "学校就业指导中心",
          "viewCount": 1234,
          "category": "招聘信息"
        }
      ],
      "total": 100,
      "page": 1,
      "pageSize": 10
    }
  }
  ```
- **功能描述**: 获取就业资讯列表，支持分页和筛选。

## 5. 响应说明

所有接口的响应格式统一为：

```json
{
  "code": 200,
  "message": "操作成功",
  "data": {}
}
```

- **code**: 响应状态码，200表示成功，其他表示失败
- **message**: 响应消息，描述操作结果
- **data**: 响应数据，根据接口不同返回不同的数据结构

## 6. 访问权限

所有就业服务接口均支持匿名访问，无需认证。
