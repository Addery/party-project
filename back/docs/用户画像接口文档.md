# 用户画像接口文档

## 1. 获取当前登录用户的画像

**接口路径**：`GET /api/user/portrait/current`

**接口说明**：获取当前登录用户的用户画像信息，包括学习标签、学习兴趣、学习偏好等。

**请求参数**：无

**响应格式**：
```json
{
  "code": 200,
  "message": "获取用户画像成功",
  "data": {
    "id": 1,
    "user": {
      "id": 1,
      "username": "zhangsan",
      "name": "张三",
      // ... 其他用户信息
    },
    "learningTags": '["资讯关注者", "学习爱好者", "活跃用户", "时事敏锐", "求知若渴"]',
    "learningInterests": '["时政要闻", "组织建设", "科技创新", "党风廉政", "基层党建"]',
    "learningPreferences": '{"newsReading":{"时政要闻":15, "组织建设":12, ...}, "learningTopics":{"理论学习":20, "党章党规":18, ...}}',
    "updatedAt": "2023-12-06T14:30:00",
    "createdAt": "2023-12-01T10:00:00"
  }
}
```

**错误响应**：
```json
{
  "code": 500,
  "message": "未登录",
  "data": null
}
```

## 2. 根据用户ID获取用户画像

**接口路径**：`GET /api/user/portrait/{userId}`

**接口说明**：根据用户ID获取指定用户的用户画像信息。

**请求参数**：
- `userId`：用户ID，路径参数，必填

**响应格式**：
```json
{
  "code": 200,
  "message": "获取用户画像成功",
  "data": {
    "id": 1,
    "user": {
      "id": 1,
      "username": "zhangsan",
      "name": "张三",
      // ... 其他用户信息
    },
    "learningTags": '["资讯关注者", "学习爱好者", "活跃用户", "时事敏锐", "求知若渴"]',
    "learningInterests": '["时政要闻", "组织建设", "科技创新", "党风廉政", "基层党建"]',
    "learningPreferences": '{"newsReading":{"时政要闻":15, "组织建设":12, ...}, "learningTopics":{"理论学习":20, "党章党规":18, ...}}',
    "updatedAt": "2023-12-06T14:30:00",
    "createdAt": "2023-12-01T10:00:00"
  }
}
```

**错误响应**：
```json
{
  "code": 500,
  "message": "用户画像不存在",
  "data": null
}
```

## 3. 手动更新当前登录用户的画像

**接口路径**：`POST /api/user/portrait/update`

**接口说明**：手动触发更新当前登录用户的用户画像信息。

**请求参数**：无

**响应格式**：
```json
{
  "code": 200,
  "message": "更新用户画像成功",
  "data": {
    "id": 1,
    "user": {
      "id": 1,
      "username": "zhangsan",
      "name": "张三",
      // ... 其他用户信息
    },
    "learningTags": '["资讯关注者", "学习爱好者", "活跃用户", "时事敏锐", "求知若渴"]',
    "learningInterests": '["时政要闻", "组织建设", "科技创新", "党风廉政", "基层党建"]',
    "learningPreferences": '{"newsReading":{"时政要闻":15, "组织建设":12, ...}, "learningTopics":{"理论学习":20, "党章党规":18, ...}}',
    "updatedAt": "2023-12-06T14:35:00",
    "createdAt": "2023-12-01T10:00:00"
  }
}
```

**错误响应**：
```json
{
  "code": 500,
  "message": "未登录",
  "data": null
}
```

## 用户画像字段说明

| 字段名 | 类型 | 说明 |
| ------ | ---- | ---- |
| id | Integer | 用户画像ID |
| user | User | 关联的用户对象 |
| learningTags | String | 学习标签（JSON格式，如：["资讯关注者", "学习爱好者"]） |
| learningInterests | String | 学习兴趣（JSON格式，如：["时政要闻", "组织建设"]） |
| learningPreferences | String | 学习偏好分析（JSON格式，包含阅读新闻和学习主题的偏好） |
| updatedAt | Date | 画像更新时间 |
| createdAt | Date | 画像创建时间 |

## 响应状态码说明

| 状态码 | 说明 |
| ------ | ---- |
| 200 | 操作成功 |
| 500 | 操作失败 |
