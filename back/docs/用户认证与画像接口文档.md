# 用户认证与画像接口文档

## 1. 用户登录接口

### 1.1 登录接口

**接口路径**：`POST /api/auth/login`

**接口说明**：用户登录，获取JWT认证令牌。

**请求头**：
```
Content-Type: application/json
```

**请求体**：
```json
{
  "username": "string",  // 用户名，必填
  "password": "string"   // 密码，必填
}
```

**响应格式**：
```json
{
  "code": 200,
  "message": "登录成功",
  "data": {
    "token": "eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...",  // JWT令牌
    "user": {
      "id": 1,
      "username": "zhangsan",
      "name": "张三",
      // ... 其他用户信息
    }
  }
}
```

**错误响应**：
```json
{
  "code": 500,
  "message": "用户名或密码错误",
  "data": null
}
```

## 2. 用户画像接口

### 2.1 获取当前登录用户的画像

**接口路径**：`GET /api/user/portrait/current`

**接口说明**：获取当前登录用户的用户画像信息，包括学习标签、学习兴趣、学习偏好等。

**请求头**：
```
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...  // JWT令牌，必填
```

**请求参数**：无

**响应格式**：
```json
{
  "code": 200,
  "message": "获取用户画像成功",
  "data": {
    "id": 1,
    "user": {
      "id": 1,
      "username": "zhangsan",
      "name": "张三",
      // ... 其他用户信息
    },
    "learningTags": '["资讯关注者", "学习爱好者", "活跃用户", "时事敏锐", "求知若渴"]',
    "learningInterests": '["时政要闻", "组织建设", "科技创新", "党风廉政", "基层党建"]',
    "learningPreferences": '{"newsReading":{"时政要闻":15, "组织建设":12, ...}, "learningTopics":{"理论学习":20, "党章党规":18, ...}}',
    "updatedAt": "2023-12-06T14:30:00",
    "createdAt": "2023-12-01T10:00:00"
  }
}
```

**错误响应**：
```json
{
  "code": 500,
  "message": "未登录",
  "data": null
}
```

## 3. 认证机制说明

### 3.1 JWT令牌获取

用户通过登录接口获取JWT令牌，令牌格式如下：
```
Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### 3.2 JWT令牌使用

所有需要认证的接口（如用户画像接口）都需要在请求头中携带JWT令牌：
```
Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
```

### 3.3 JWT令牌有效期

默认令牌有效期为24小时，可以在`application.yaml`中配置：
```yaml
jwt:
  expiration: 86400000  # 令牌有效期（毫秒）
```

## 4. 接口调用示例

### 4.1 登录示例

```bash
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"username": "Addery", "password": "password123"}'
```

### 4.2 获取当前用户画像示例

```bash
curl -X GET http://localhost:8080/api/user/portrait/current \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### 4.3 根据用户ID获取用户画像示例

```bash
curl -X GET http://localhost:8080/api/user/portrait/6 \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

### 4.4 更新用户画像示例

```bash
curl -X POST http://localhost:8080/api/user/portrait/update \
  -H "Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9..."
```

## 5. 响应状态码说明
