# 主题活动相关接口规范

## 1. 接口概述

本接口规范定义了党建管理系统中主题活动相关的所有接口，包括活动分类、活动列表、活动详情、活动报名以及首页推荐活动等功能。

## 2. 基础信息

- 接口前缀：`/api`
- 响应格式：JSON
- 数据传输方式：HTTP
- 认证方式：JWT Token（部分接口需要）

## 3. 接口列表

| 接口名称 | 请求方式 | 接口路径 | 功能描述 | 是否需要认证 |
|---------|---------|---------|---------|------------|
| 获取活动分类 | GET | /activities/categories | 获取所有活动分类列表 | 否 |
| 获取活动列表 | GET | /activities | 获取活动列表，支持分页、分类筛选和关键词搜索 | 否 |
| 获取活动详情 | GET | /activities/{id} | 获取指定活动的详细信息 | 否 |
| 活动报名 | POST | /activities/{id}/register | 报名参加指定活动 | 是 |
| 获取首页推荐活动 | GET | /home/activities | 获取首页推荐的活动列表 | 否 |

## 4. 详细接口说明

### 4.1 获取活动分类

**接口路径**：`GET /api/activities/categories`

**功能描述**：获取系统中所有可用的活动分类列表。

**请求参数**：无

**响应示例**：

      "name": "主题党日"

      "name": "志愿服务"
    },
    {

### 4.2 获取活动列表

**接口路径**：`GET /api/activities`

**功能描述**：获取活动列表，支持分页、分类筛选和关键词搜索。

**请求参数**：

| 参数名称 | 类型 | 是否必填 | 描述 | 默认值 |
|---------|------|---------|------|--------|
| page | Integer | 否 | 当前页码 | 1 |
| pageSize | Integer | 否 | 每页记录数 | 10 |
| category | String | 否 | 活动分类ID | 无 |
| keyword | String | 否 | 搜索关键词（标题或描述） | 无 |

**响应示例**：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "total": 100,
    "pages": 10,
    "page": 1,
    "pageSize": 10,

        "date": "2025-12-15",
        "location": "党建活动室",
        "category": "theme",
        "status": "published",
        "description": "深入学习党的二十大精神，提升党员政治素养",
        "participants": 15,
        "image": "https://picsum.photos/id/100/800/450"
      },
      // 更多活动...
    ]
  }
}
```

### 4.4 活动报名

**接口路径**：`POST /api/activities/{id}/register`

**功能描述**：用户报名参加指定活动。

**请求参数**：

| 参数名称 | 类型 | 是否必填 | 描述 |
|---------|------|---------|------|
| id | Integer | 是 | 活动ID（路径参数） |
| userId | Integer | 是 | 用户ID |

**请求示例**：
```json
{
  "userId": 1
}
```

**响应示例**：
```json
{
  "code": 200,
  "message": "操作成功",
  "data": {
    "activityId": 1,
    "userId": 1,
    "registerTime": "2025-12-10T10:30:00",
    "status": "registered"
  }
}
```

**错误响应示例**：
```json
{
  "code": 500,
  "message": "活动不存在",
  "data": null
}
```

**可能的错误信息**：
- 活动不存在
- 已报名参加该活动
- 活动人数已满

### 4.5 获取首页推荐活动

**接口路径**：`GET /api/home/activities`

**功能描述**：获取首页推荐的活动列表。

**请求参数**：

| 参数名称 | 类型 | 是否必填 | 描述 | 默认值 |
|---------|------|---------|------|--------|
| limit | Integer | 否 | 返回记录数 | 6 |
| sortBy | String | 否 | 排序字段（date, participants） | date |
| order | String | 否 | 排序方式（asc, desc） | desc |

**响应示例**：
```json
{
  "code": 200,
  "message": "获取成功",
  "data": [
    {
      "id": 1,
      "title": "学习贯彻党的二十大精神主题党日活动",

      "location": "党建活动室",
      "participants": 15,
      "status": "published",
      "image": "https://picsum.photos/id/100/800/450",
      "tags": ["主题党日"]
    },
    // 更多推荐活动...
  ]
}
```

## 5. 数据模型定义

### 5.1 活动分类DTO (ActivityCategoryDTO)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| id | String | 分类ID |
| name | String | 分类名称 |

### 5.2 活动列表项DTO (ActivityItemDTO)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| id | Integer | 活动ID |
| title | String | 活动标题 |
| date | String | 活动日期 |
| location | String | 活动地点 |
| category | String | 活动分类 |
| status | String | 活动状态 |
| description | String | 活动描述 |
| participants | Integer | 当前参与人数 |
| image | String | 活动封面图片 |

### 5.3 活动列表DTO (ActivityListDTO)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| total | Long | 总记录数 |
| pages | Integer | 总页数 |
| page | Integer | 当前页码 |
| pageSize | Integer | 每页记录数 |
| activities | List<ActivityItemDTO> | 活动列表 |

### 5.4 活动详情DTO (ActivityDetailDTO)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| id | Integer | 活动ID |
| title | String | 活动标题 |
| date | String | 活动日期 |
| location | String | 活动地点 |
| category | String | 活动分类 |
| status | String | 活动状态 |
| description | String | 活动描述 |
| content | String | 活动内容 |
| participants | Integer | 当前参与人数 |
| capacity | Integer | 最大参与人数 |
| image | String | 活动封面图片 |
| attachments | List<AttachmentDTO> | 活动附件列表 |

### 5.5 活动报名请求DTO (ActivityRegistrationRequest)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| userId | Integer | 用户ID |

### 5.6 活动报名响应DTO (ActivityRegistrationResponse)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| activityId | Integer | 活动ID |
| userId | Integer | 用户ID |
| registerTime | LocalDateTime | 报名时间 |
| status | String | 报名状态 |

### 5.7 首页活动项DTO (HomeActivityItemDTO)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| id | Integer | 活动ID |
| title | String | 活动标题 |
| type | String | 类型（固定为"活动"） |
| date | String | 活动日期 |
| location | String | 活动地点 |
| participants | Integer | 当前参与人数 |
| status | String | 活动状态 |
| image | String | 活动封面图片 |
| tags | List<String> | 活动标签 |

### 5.8 附件DTO (AttachmentDTO)

| 字段名 | 类型 | 描述 |
|-------|------|------|
| id | Integer | 附件ID |
| name | String | 附件名称 |
| url | String | 附件URL |

## 6. 错误码定义

| 错误码 | 描述 |
|-------|------|
| 200 | 成功 |
| 500 | 服务器内部错误 |
| 400 | 请求参数错误 |
| 401 | 未授权 |
| 404 | 资源不存在 |

| 状态值 | 描述 |
|-------|------|

## 8. 安全规范

1. 所有需要认证的接口必须在请求头中携带有效的JWT Token
2. 敏感操作（如活动报名）需要进行权限验证
3. 输入参数需要进行严格的验证和过滤，防止SQL注入和XSS攻击
4. 活动报名时需要进行重复报名检查和人数限制检查
