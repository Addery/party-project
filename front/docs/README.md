# 党建管理系统

## 项目概述

党建管理系统是一个基于Vue 3开发的现代化党建工作平台，旨在为党组织提供高效、便捷的管理工具，提升党建工作的信息化水平和工作效率。系统分为前台展示和后台管理两大模块，涵盖了党建工作的各个方面，包括新闻动态、通知公告、学习资源、党员管理、组织活动等功能。

## 技术栈

- **前端框架**：Vue 3
- **构建工具**：Vite
- **状态管理**：Pinia
- **路由管理**：Vue Router 4
- **UI组件库**：Element Plus
- **HTTP客户端**：Axios
- **图标库**：@element-plus/icons-vue
- **Markdown解析**：marked
- **后端支持**：Express (内嵌简单服务)

## 功能模块

### 前台展示模块

1. **首页**：展示近期新闻动态、通知公告、主题活动和样板支部等内容
2. **新闻列表/详情**：查看党建新闻和详细内容
3. **学习资源**：提供党建学习资料和资源下载
4. **活动列表**：展示党组织开展的各项活动
5. **关于我们**：介绍党组织的基本情况
6. **通知公告**：查看党组织发布的通知和公告
7. **个人中心**：党员个人信息管理和学习记录
8. **登录/注册**：用户身份认证和注册

### 后台管理模块

1. **仪表盘**：展示系统运行状态和关键数据
2. **用户管理**：管理系统用户和权限
3. **新闻管理**：发布和管理党建新闻
4. **组织管理**：管理党组织结构和信息
5. **三会一课**：管理支部会议、党课等活动
6. **主题党日**：组织和管理主题党日活动
7. **党员发展**：跟踪和管理党员发展流程
8. **党员管理**：管理党员基本信息和组织关系
9. **党费收缴**：管理党费缴纳记录和统计
10. **学习资源**：上传和管理学习资源
11. **考试管理**：组织在线考试和评估
12. **数据分析**：党建工作数据统计和分析
13. **互动管理**：管理党员互动和交流
14. **系统设置**：配置系统参数和功能
15. **教育管理**：管理党员教育计划和记录
16. **党建活动**：统筹管理各类党建活动

## 安装与运行

### 安装依赖

```bash
npm install
```

### 开发模式运行

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

### 前台组件

### 后台组件

- **AdminLayout.vue**：后台管理布局

## 路由配置

系统采用Vue Router进行路由管理，将前台和后台路由分离，实现了权限控制和页面导航功能。

## 状态管理

使用Pinia进行状态管理，将应用状态集中管理，提高代码的可维护性和可测试性。

## 响应式设计

系统采用响应式设计，适配不同屏幕尺寸的设备，提供良好的用户体验。

## 安全性

- 用户身份认证和授权
- 数据加密传输
- 防止XSS和CSRF攻击
- 输入验证和数据过滤
