# 文档目录说明

本目录包含党员教育与管理系统的所有文档文件，按功能模块和文档类型进行分类整理。

## 文档列表

### 项目说明文档
- **README.md** - 项目主说明文档，包含项目概述、技术栈、功能模块、安装运行步骤等信息
- **BACKEND_README.md** - 后端开发说明文档

### API接口规范
- **API_SPEC.md** - 项目API接口总规范
- **employment_api_design.md** - 就业相关API设计文档
- **headline_news_api_document.md** - 头条新闻API文档
- **home_activity_detail_api_spec.md** - 首页活动详情API规范
- **home_and_activities_api_spec.md** - 首页和活动API规范
- **learning_api_spec.md** - 学习相关API规范
- **learning_resource_api_specification.md** - 学习资源API规范（前台）
- **model_branch_api_spec.md** - 模范支部API规范
- **news_api_document.md** - 新闻API文档
- **news_api_specification.md** - 新闻动态API规范
- **notice_api_specification.md** - 通知API规范

### 需求文档
- **related_news_requirements.md** - 相关新闻需求文档

## 文档使用说明

1. 项目整体情况请查看 `README.md`
2. API接口相关文档按功能模块分类，可根据需要选择查看
3. 需求文档包含业务需求和功能说明
